# YODA

Multiple repositories management.

## Installation

Available for Python 2.x & 3.x

Install yoda:
`$ ./setup.py install`

Uninstall yoda:
`$ pip uninstall yoda`

## Running tests
In project root directory, run following command to

To run unittests:
`$ nosetests`

To check code style:
`$ pep8 ./`
